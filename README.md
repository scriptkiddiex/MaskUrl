# MaskUrl  (v1.0.0)
**Developed by: ScriptKiddie**

MaskUrl is a professional-grade Python utility designed for URL masking. It leverages shortening services and social engineering techniques to transform suspicious links into trustworthy-looking URLs for security awareness training and authorized penetration testing.

---

##  Features
* **Multi-Engine Support:** Seamless integration with TinyURL, Da.gd, and Clck.ru.
* **Advanced Domain Masking:** Custom `@` symbol manipulation to overlay trusted domains (e.g., google.com, facebook.com).
* **Contextual Keywords:** Inject phishing-simulation keywords (e.g., `login`, `verify-account`, `free-credit`) to increase the effectiveness of security tests.
